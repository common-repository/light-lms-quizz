Light LMS Quizz provides Gutenberg blocks for multiple choice questions. 

== Description ==

Provide for interactive questions so your puplis will better get your lessons and learn by answering questions. With 
Light LMS Quizz you can add interactive questions to lessons. Light LMS Quizz provides the blocks for entering 
multiple choice questions and their answers and evaluates the answers given by users or visitors without any waiting.
Simply choose a block like the block for a multiple choice question with five answers and leave the superfluos answer 
blank to make it a multiple choice question with four answers. Enter the text on the button, feedback when right and 
feedback when wrong in the block settings column to the right and your are all set. Optionally you can choose 
background and text colors for the feedback and markup the question with bold and italic.

Multple choice questions can have one or multiple answers by selecting the appropriate setting. Enter the right answer
as a letter or multiple letters. Optionally you can use digits starting with 1 for the first possible choice.

== Installation ==
 
1. Install the plugin via Plugins -> New plugin. 
2. Activate the Light LMS Quizz plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
 
= Can I use Light LMS Quizz with another LMS? =

Yes you can. Light LMS Quizz can make your lessons in WordPress pages or posts interactive by giving excercise 
questions. Answering the question is folowed by a feedback to your liking.

= Can I make an exam with Light LMS Quizz =
 
No. Use any other LMS for exams.
